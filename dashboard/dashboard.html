    How to read this

    Lowering the threshold flags more patients as high-risk — recall improves (fewer missed cases) but precision drops (more false alarms). Raising it does the opposite. Clinical screening tools typically favor a lower threshold to avoid missing true positive cases, accepting more false positives as the cost of caution.
